Guld mail is a signed, encrypted, and witnessed email system.

### Install

##### Node

```sh
npm i guld-mail
```

### Usage

##### Node

```
const { getMail } = require('guld-mail')
```

##### CLI

See [guld-mail-cli](https://bitbucket.org/isysd/tech-js-node_modules-guld-mail-cli).
